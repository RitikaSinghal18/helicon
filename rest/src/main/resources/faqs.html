What is displayed when I click "Home?"
The home page displays various details about the corpus. For instance, statistics on the categories to browse through along with items like minimum, average, and maximum count of projects in the corpus.

What is displayed when I click "Browse?"
Browse allows you to navigate starting with available categories:
  • Clicking a category displays a list of that category (e.g., list of languages)
  • Clicking a specific category item displays a list of projects (e.g., list of java projects)
  • Clicking a specific project metadata link displays the stored metadata (project count, crawled date, etc.
What is displayed when I click "Advanced Search?"
Advanced search is not yet implemented.
